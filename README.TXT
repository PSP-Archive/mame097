実行：
■バイナリは別ファイルです。
■menu.bmpという名前で480x272x24bppのファイル（無圧縮ＢＭＰ形式．383kBytes）
を、EBOOT.PBPと同じ場所に置くと壁紙になります。
================================================
終了方法：
★pmame04互換の[start]+[L]+[R]でも終われる筈ですが、不安定。
★[R]でポーズ中に、[L]のMAMEメニューから[End This Emulator]を選び、
その後[R]を押すと終われる筈です。しかしこの方法でも死ぬ事がある。
何故だか解からないです。
================================================
突然死には「電源スイッチ長押し」で対処して下さい。
つまりマダそんなレベルです。

この「突然死」はPSPは「落ちない」様ですので、「メモリ違反」ではなく、
「無限ループ」の様です。このバグが無けりゃ結構イケるんだケドなあ...。 orz

註：メモリ違反：もしメモリ違反なら強制電源断の筈。

開発：
■そのままコンパイルすれば、namcoNA_A10が出来ます。

makefileを

#TARGET = namcoNA
#TARGET = taitoAX
#TARGET = m72
 TARGET = namcos86

#################↑ここを変えるのでっす。

へ変えてコンパイルすれば、namcos86_A03が出来ます。
